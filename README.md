# BagFolio
Simple crypto portfolio tracker.

## Requirement
- [json-simple](https://github.com/fangyidong/json-simple) A simple Java toolkit for JSON.

## Supported Exchanges
- Bittrex
- Binance
- Poloniex
- Kucoin
